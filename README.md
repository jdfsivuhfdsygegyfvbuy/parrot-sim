# Parrot Sim
### Short description
This is a parrot simulator, made in **C# .NET** for **Windows**.
### Features
- Accurate parrot simulation
- Fun features
- Command-line controls
- Source code available (open-source).
### How to build
#### Perquisites:
- Dotnet SDK (7.0 or higher).
- Windows 10 or 11.
- A great sense of humor.
#### Commands to run:
```cmd
git clone https://github.com/jdfsivuhfdsygegyfvbuy/parrot-sim-net
cd parrot-sim-net
dotnet build
```
*Then run it with:* `.\bin\Release\parrotsim.exe`
